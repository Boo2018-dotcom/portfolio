Design Reflection

Constraints don't
limit design.
They define it.

Fitting a complete UPI payment flow into 390px taught me more about interaction design than any full-canvas project ever has.

→
If it's not essential, it's a distraction Every element had to justify its presence on screen
→
Trust is built in the first tap, not the last On a small screen, confidence must arrive instantly
→
"What if the user wants to...?" → Remove it Edge cases don't belong on a glanceable device
The cleanest interface I've ever shipped came from the most constrained canvas I've ever worked on.
System UI
